#reactForWebpack

这是一个基于webpack的reactjs项目

### 特点：
* 基于webpack进行编译
* 使用了Reactjs进行开发
* 样式使用SCSS进行开发
* 通过webpack的server服务器作为本地的开发服务器

### 使用说明：

初始化项目
> npm install

开发调试

> npm run dev

编译部署

> npm run build

### 项目配置

在项目中的相关配置全部在webpack.config.js文件中进行设置，具体参数可查看注释。
